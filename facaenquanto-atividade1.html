Estrutura FACA-ENQUANTO-Atividade1

Esta atividade utiliza a estrutura de repetição FAÇA-ENQUANTO (do...while) para somar números informados pelo usuário. O programa executa o bloco pelo menos uma vez, acumula os valores digitados e só encerra quando o usuário digitar 0, exibindo no console a soma final.


Utilize console.log (Ctrl+Shift+J) para visualizar os resultados
