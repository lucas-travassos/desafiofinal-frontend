Estrutura VETORES BIDIMENSIONAIS-Atividade2

Esta atividade utiliza uma matriz 2x2 para armazenar valores digitados pelo usuário. Após preencher a tabela, o programa permite escolher uma linha específica e exibe a soma dos elementos dessa linha, junto com a matriz original.


Utilize console.log (Ctrl+Shift+J) para visualizar os resultados
