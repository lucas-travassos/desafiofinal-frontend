Estrutura CASO-Atividade2

Esta atividade propõe a criação de uma calculadora em JavaScript utilizando a estrutura condicional CASO (switch). O programa valida o preenchimento dos campos, realiza operações básicas (soma, subtração, multiplicação e divisão), impede a divisão por zero e exibe no console o resultado, informando também se ele é par ou ímpar.


Utilize console.log (Ctrl+Shift+J) para visualizar os resultados
